# Shopping Cart  Example

This is a simple e-commerce demo built with **React** and **Redux Toolkit**. It features a list of products, a shopping cart, and fully interactive add/remove functionality. The UI is styled with **Tailwind CSS** and supports scrollable product and cart sections with hidden scrollbars.

---

## Features:-
- Display a list of products with **name, price**.
- Add products to the cart.
- Remove products from the cart.
- Clear the cart with a single button.
- Display **total cart value**.
- Scrollable product and cart sections with hidden scrollbars.
- Responsive layout using Tailwind CSS.
- Styled buttons with hover effects.

---

## Technologies Used
- **React 18**
- **Redux Toolkit**
- **React-Redux**
- **Tailwind CSS**
- **JavaScript (ES6+)**
